# 02 — Manage Projects and Insight Sources

**What to build:** The Project Library and Project workspace entry flow, including project creation, loading, renaming, duplication, deletion, Insight Source text entry, autosave, and convenient plain-text or Markdown import.

**Type:** feature

**Blocked by:** 01

**Status:** ready-for-agent

- [ ] Users can create, load, rename, duplicate, and delete locally saved Projects.
- [ ] Each Project contains exactly one workflow chain and one Insight Source text block.
- [ ] Insight text autosaves and can be imported from `.txt` or `.md` files.
- [ ] Destructive actions are clearly confirmed and preserve unrelated Projects.
- [ ] Project and Insight behavior is covered through public service and UI seams.
