# 03 — Configure the shared workflow safely

**What to build:** A global workflow configuration experience for the three stage prompts and stage-specific OpenAI model choices, with large-draft editing, protected placeholders, live model discovery, explicit save, reset to bundled defaults, and configuration import/export.

**Type:** feature

**Blocked by:** 01

**Status:** ready-for-agent

- [ ] One saved prompt and model configuration is shared across all Projects.
- [ ] Prompt drafts can be edited comfortably, compared with the saved version, explicitly saved, or discarded.
- [ ] Required placeholders cannot be deleted or malformed by prompt editing.
- [ ] Available compatible text and image models are obtained from OpenAI and filtered for the relevant stage.
- [ ] Image quality can be selected independently of the image model.
- [ ] Users can reset to bundled defaults and export or import the global configuration.
