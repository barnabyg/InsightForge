# 05 — Generate a coordinated Concept Screen Set

**What to build:** Generate exactly three coordinated mid-fidelity concept screens from the Design Brief, sequentially and with prior screens used as references, while exposing progress, validation, usage, resumable partial failure, and a focused gallery.

**Type:** feature

**Blocked by:** 04

**Status:** ready-for-agent

- [ ] Running the stage produces exactly three PNG concept screens from the current Design Brief.
- [ ] Screen two references screen one, and screen three references both earlier screens.
- [ ] The first screen establishes dimensions and later screens use matching fixed dimensions.
- [ ] A failed or cancelled partial set can be resumed without regenerating valid completed screens.
- [ ] The gallery supports accessible inspection of each screen and its provenance.
- [ ] Image structure, progress, timing, and usage are reported through the Run Inspector.
